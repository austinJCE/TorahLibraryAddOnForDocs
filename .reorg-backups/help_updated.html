User Guide

The add-on now centers around two tabs: Texts for source finding and insertion, and Voices for source sheets. In Texts, search and insert first; open More Options only when you want to override your saved defaults for the current insert.

Texts Voices Search Preview Insert

Quick start

  1. Open Texts for standard sources or Voices for source sheets.
  2. Type a reference, title, phrase, or paste a Sefaria URL.
  3. Click Search.
  4. Select the result you want.
  5. Review the preview, then click Add Source.
Open in Sefaria follows your current display mode: Hebrew only opens in Hebrew, translation only opens in English, and bilingual opens in bilingual view.

Texts vs. Voices

Texts

Use this tab to find and insert standard Sefaria texts. This is where display, layout, versions, preview, and More Options live.

Voices

Use this tab to search source sheets. Text-only controls stay hidden here so the workflow remains focused.

What is saved for you

Your settings are split into two layers:

Preferences

Your cross-document defaults. Use these for the way you usually like to work.

Session Settings in More Options

At the top of More Options you can temporarily override defaults for the current session, then reset or save them as your new defaults.

Search filters like corpus, translation language, and sort are remembered too.

What lives in More Options

Session Settings

Reset or save the current Texts-session overrides.

Display & Layout

Choose Hebrew only, translation only, or both. When both are shown, you can choose stacked or side-by-side bilingual layout.

Interactive Hebrew preview

Toggle vowels and cantillation live, choose transliteration, and see the status line update before insertion.

Text details

Line markers, translation details, transliteration, and Sefaria link behavior all live here.

How result filtering works

After you search, the result list becomes its own temporary workspace.

  • You can filter by translation language under Library matches.
  • You can hide corpus groups from the current result set.
  • You can restore a hidden corpus with the restore control.
These removals are temporary and last only until the next search.

Preview and versions

  • The preview shows what will be inserted using your current display settings.
  • Translation versions appear when more than one translation exists.
  • Original Versions appear only when more than one original/source version exists.
  • Translation details adds version/source information below inserted translation text when enabled.

Session Library / History

The history button sits to the left of the search bar. Use it to reopen recent selections and pinned items.

  • Pin a result to keep it in Session Library.
  • Recent items and pinned items are grouped in the history modal.
  • Pinned items sort above recent items within their grouping.

When to use Preferences instead

Open Preferences when you want to change your usual defaults. Use More Options for one-off or session-level decisions inside Texts.
