User Guide

The sidebar is designed to stay simple by default. Search and insert first; open More Options only when you want to override your saved defaults for a single insert.

Quick start

  1. Type a reference, title, phrase, or paste a Sefaria URL.
  2. Click Find.
  3. Select the result you want.
  4. Review the preview.
  5. Click Insert.
The Open on Sefaria button follows the sidebar display mode: Hebrew only opens in Hebrew, translation only opens in English, and bilingual opens in bilingual view.

What is saved for you

Your settings are split into two layers:

Preferences

Your cross-document defaults. Use these for the way you usually like to work.

More Options

Your current insert choices. These let you override defaults without changing the basic flow.

Search filters like corpus, translation language, sort, and relevance are also remembered for you across documents.

What lives in More Options

Use More Options when you need to change how the inserted source should look, or when you want to narrow the search.

Display

Hebrew only, translation only, or both. If both are shown, you can also choose the bilingual layout.

Text and links

Vowels, cantillation, line markers, transliteration, translation details, and whether to insert a Sefaria link.

Search filters

Corpus filters, translation-language filter, sort mode, and relevance behavior.

Versions

Choose a translation version when needed, and optionally a specific Hebrew/source edition in the advanced section.

How result filtering works

After you click Find, the result list becomes its own temporary workspace.

  • You can remove whole corpus groups from the current result set.
  • You can remove active corpus chips from the current result set.
  • These removals are temporary and last only until the next time you click Find.
Think of these removals as quick cleanup for the current search, not permanent preference changes.

Preview behavior

The preview shows what will be inserted using your current display settings.

  • Hebrew only: preview shows Hebrew only.
  • Translation only: preview shows translation only.
  • Both: preview shows both according to the selected bilingual layout.
  • If transliteration is enabled, it appears directly under the Hebrew text when Hebrew is included.

When to use Preferences instead

Open Preferences when you want to change your usual defaults. Use More Options when you want to make a one-off decision for the current search or insert.

For bug reports, support, or feature requests, use the separate Support window rather than this guide.

AI Lesson Generator: Keys, Privacy, Permissions, and Modes

Basic Mode

Uses the Merkaz-managed AI route on the website. Users do not need to choose a provider or model, and the site can change routing, prompts, cooldowns, and orchestration without updating the add-on.

Advanced Mode

Power users can override the route, choose a model when allowed, and optionally use a request-level API key or a saved key configured in Preferences.

API Key Policy

Keys pasted into the AI modal are intended for that request only unless a separate save-key flow is explicitly used in Preferences. Do not include sensitive or confidential information in prompts.

Google Permission Note

Google may describe the add-on as able to “run when you are not present” because it can make secure external requests to the Merkaz site. AI requests are only sent when you choose an add-on action; this project does not rely on background polling or time-based triggers for AI generation.

AI-generated content should always be reviewed, edited, and fact-checked before teaching, publishing, or sharing.
