About

Torah Library Add-on for Docs · UX and maintenance fork

This add-on helps you find, preview, and insert Sefaria sources into Google Docs with cleaner defaults, safer insertion behavior, and improved bilingual workflows.

Contact support GitHub repository Buy me a coffee

What’s in this fork

  • Unified Find & Insert sidebar flow
  • Improved display and bilingual layout controls
  • Normalized translation/source version selection
  • Persistent preferences across documents
  • Safer insertion pipeline for richer formatting

Version and changelog

Current active line: 2.0 fork stream

Recent work has focused on sidebar simplification, transliteration support, result refinement, and preference ergonomics.

For the full running changelog, see the release notes included with the add-on codebase.

For technical users

The project is maintained in Apps Script/HTML/CSS/JS and is easiest to improve through concrete bug reports, screenshots, and reproducible test cases.

A direct in-dialog support form with optional document context can be added later, but is not part of this pass.
