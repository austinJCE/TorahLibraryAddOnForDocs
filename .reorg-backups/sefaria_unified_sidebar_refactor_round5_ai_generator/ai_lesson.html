AI Lesson Generator

Generate a structured source-sheet or shiur draft directly into the active Google Doc. Review, fact-check, and edit before sharing.

Managed-key mode is intended for light internal use. This build enforces a cooldown when the add-on is using behind-the-scenes keys. If no managed key is available, paste your own or save a provider-specific key.
Supported provider families in this build: ChatGPT via OpenAI Responses, Claude via Anthropic Messages, and Gemini via Google’s generateContent endpoint.
