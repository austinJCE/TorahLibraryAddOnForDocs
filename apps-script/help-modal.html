Most users should spend their working time in Texts or Voices. Use Preferences to set defaults, not for ordinary day-to-day insertion.

Texts: the standard workflow

  1. Open Texts.
  2. Type a reference, title, phrase, or paste a Sefaria URL.
  3. Run the search and select the result you want.
  4. Review language layout, display layout, translation choice, and Preview.
  5. Insert once Preview matches what you want in the document.

Language layout

This is a live working control and should stay easy to access in Texts.

Display layout

Use this to change how the inserted content is arranged before you insert.

Translation selection

When multiple translations exist, choose directly in the sidebar. In Translation only or Hebrew on top layout, hold Ctrl / Cmd to select multiple versions — each will be inserted in sequence.

Preview

Preview should show what will be inserted right now.

Voices: the source-sheet workflow

  1. Open Voices.
  2. Search for the sheet or source-sheet content you want.
  3. Select the result and review the available preview or metadata.
  4. Insert or use the relevant Voices action.

Voices is intentionally narrower than Texts. Text-specific controls should stay out of the way here so the source-sheet workflow remains focused.

Text & Links: current-session overrides

Text & Links is for temporary overrides inside Texts. Use it when you want to change citation, transliteration, niqqud, or cantillation for the current session without changing your saved defaults.

Source citation

Turn the source citation block below the inserted text on or off for the current session.

Transliteration

Include transliteration when needed without permanently changing your defaults.

Niqqud

Match the Hebrew display details you want to insert.

Cantillation

Use the viewer controls and confirm the result in Preview before inserting.

Changes in Text & Links should update Preview immediately. They affect the current session unless you explicitly save them as new defaults.

Preferences vs. current session

Preferences

Your saved defaults. These populate Texts when you start a session.

Current session

Your live choices in the sidebar right now. Preview follows these, not just your saved defaults.

Save as defaults

Use this only when you want your current session choices to become your new starting defaults.

Reset to defaults

Return the current session to your saved Preferences.

Troubleshooting

A toggle is not responding

Reopen the section, then reopen the add-on. Refresh the document if the toggle still looks stuck.

The output did not match the setting

Check whether another setting, workspace, or current-session override changed the insertion state.

The sidebar layout looks wrong

Compressed or unstable layout is often fixed by reopening the add-on or refreshing the document.

Things look old after an update

This is often stale state rather than a broken setting. Refresh first, then test again.

What to include in a bug report

If you email support, include enough detail to make the issue reproducible.

  • which workspace you were using: Texts, Voices, or Preferences
  • what you searched for or selected
  • which settings or overrides were active
  • what you expected to happen
  • what happened instead
  • whether reopening the add-on or refreshing the document changed the result
Screenshots are especially helpful for layout issues, Preview mismatches, and setting-sync problems.

When to email support

Email support when the self-serve steps do not resolve the issue, when you have a reproducible bug, or when you need advanced guidance that is not covered in Help.
